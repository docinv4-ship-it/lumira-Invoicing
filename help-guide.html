Troubleshooting

Quick fixes for the most common issues.

Help
Preview is blank
Select a template and fill the main fields first. The preview needs at least one template and basic data to render properly.
PDF does not match preview
Keep text short and avoid very long notes. Excess content can change spacing and affect the final exported layout.
Logo is missing
Upload a PNG or WEBP and make sure the file size is reasonable. Transparent logos usually look the cleanest.
